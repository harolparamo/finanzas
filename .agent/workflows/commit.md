---
description: Sincroniza los cambios locales con el repositorio de GitHub (add, commit, push)
---

Este workflow automatiza la subida de cambios al repositorio.

### Pasos:

1. **Preparar archivos**
   Ejecuta `git add .` para incluir todos los cambios.

2. **Crear commit**
   Ejecuta `git commit -m "[tu mensaje aquí]"` (Reemplaza el texto entre corchetes con tu descripción).

3. **Subir al repositorio**
// turbo
   Ejecuta `git push origin main` para enviar los cambios a GitHub.
